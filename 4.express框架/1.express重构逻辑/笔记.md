## HTTP 响应状态码

| 状态码 | 含义       |
| ------ | ---------- |
| 1XX    | 信息性     |
| 2XX    | 成功       |
| 3XX    | 重定向     |
| 4XX    | 客户端错误 |
| 5XX    | 服务器错误 |
